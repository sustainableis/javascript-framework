# SIS Framework - AngularJS
Javascript Framework for module discovery and orchestration.

## Installation

```bash
npm install
npm install -g bower
bower install
npm install -g grunt-cli
```

## Build
```bash
grunt concat:dist
grunt uglify:dist
```

or

```bash
grunt
```
